HR SYSTEM PROJECT

Good day. My project is an HR Management System developed using PHP, MySQL, Bootstrap, HTML, CSS, and JavaScript.

The system is designed to help organizations manage employees, payroll, attendance, leave requests, tasks, and employee records digitally.

The system supports three main user roles:

* Admin
* HR
* Employee

Each role has different permissions and dashboard access.


Developer: Ifeanyi Favour

FEATURES:
- Admin, HR, Employee roles
- Leave management
- Attendance tracking
- Payroll system
- Task assignment
- Profile management

HOW TO RUN:
1. Import hr_suite.sql into phpMyAdmin
2. Place project in htdocs
3. Start XAMPP (Apache + MySQL)
4. Visit: http://localhost/HR_SUITE_PROJECT
